# 🐸 ゲロゲロ! — Ribbit! 🐸

Ribbit aims to be a fast, slim and unique Nostr experience, similar to Reddit.

Users will have the freedom to create and browse their own feeds, curated to their exact preferences.

| Compare | Reddit | Ribbit |
| ------- | ------ | ------ |
| Custom Feeds  | ✔️  | ✔️  |
| Decentralized | ❌  | ✔️  |
| FOSS  | ❌  | ✔️  |
| Private  | ❌  | ✔️  |
